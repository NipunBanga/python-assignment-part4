# Product Explorer & ML Visualization (Part 4)

## Overview
This project demonstrates data analysis, visualization, and basic machine learning using Python.

## Features
- Data loading and inspection using pandas
- Data visualization using matplotlib and seaborn
- Feature engineering (average score calculation)
- Multiple plots:
  - Bar chart
  - Histogram
  - Scatter plot
  - Box plot
  - Line chart
- Machine Learning using Logistic Regression
- Train-test split and accuracy evaluation

## Files Included
- part4_visualization_ml.ipynb
- students.csv
- README.md

## How to Run
1. Open the notebook in Jupyter Notebook or Google Colab
2. Upload students.csv if using Colab
3. Run all cells (Runtime → Run All)

## Output
- Visual graphs for data analysis
- Model training and test accuracy

## Concepts Used
- Pandas (data handling)
- Matplotlib & Seaborn (visualization)
- Scikit-learn (ML model)
- Train-test split & scaling
